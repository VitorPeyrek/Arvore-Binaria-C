# Arvore-Binaria---C
Este projeto foi desenvolvido como parte da avaliação AVA 2 da disciplina de Estrutura de Dados da Universidade Veiga de Almeida (UVA).

O objetivo da atividade foi implementar uma Árvore Binária de Busca (ABB) utilizando a linguagem C, aplicando conceitos fundamentais estudados ao longo da disciplina, como estruturas dinâmicas, recursividade, manipulação de ponteiros e gerenciamento de memória.

A Árvore Binária de Busca é uma estrutura de dados não linear amplamente utilizada para organizar informações de forma eficiente, permitindo operações rápidas de inserção, busca e navegação entre os elementos armazenados.

Objetivos da Atividade
Implementar uma Árvore Binária de Busca (ABB);
Aplicar conceitos de recursividade;
Utilizar alocação dinâmica de memória;
Desenvolver operações de inserção e busca de elementos;
Implementar percursos em árvores;
Compreender a organização hierárquica de dados.

Durante o desenvolvimento do projeto foram utilizados os seguintes conceitos:

Estruturas de Dados;
Árvores Binárias de Busca;
Ponteiros;
Recursividade;
Alocação Dinâmica de Memória;
Funções Recursivas;
Organização Hierárquica de Dados;
Linguagem C;
