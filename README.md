# Arvore-Binaria---C
Esta atividade foi desenvolvida com o objetivo de aplicar os conceitos fundamentais de Estruturas de Dados utilizando a linguagem C. O projeto consiste na implementação de uma Árvore Binária de Busca (ABB), uma estrutura de dados não linear amplamente utilizada para armazenamento, organização e recuperação eficiente de informações.

Durante o desenvolvimento foram aplicados conceitos importantes da disciplina, como alocação dinâmica de memória, manipulação de ponteiros, funções recursivas e organização hierárquica de dados.

Objetivos
Compreender o funcionamento de uma Árvore Binária de Busca;
Implementar operações de inserção e busca de elementos;
Utilizar recursividade para percorrer a estrutura da árvore;
Aplicar conceitos de gerenciamento de memória utilizando ponteiros;
Desenvolver uma aplicação interativa baseada em menu.

Sobre a Estrutura de Dados

A Árvore Binária de Busca (ABB) é uma estrutura hierárquica composta por nós. Cada nó possui um valor armazenado e referências para seus nós filhos.

As regras de organização da ABB são:

Valores menores que o nó atual são armazenados à esquerda;
Valores maiores que o nó atual são armazenados à direita;
Não são inseridos valores duplicados.

Essa organização permite realizar buscas de forma mais eficiente quando comparada a estruturas lineares em determinadas situações.
